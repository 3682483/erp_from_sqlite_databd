# ERP 管理系统

基于 SQLite 的企业资源计划（ERP）管理系统，用于企业日常运营管理。

## 功能模块

- **基础数据管理**：客户、供应商、物料等基础信息维护
- **采购管理**：采购订单、入库管理、供应商管理
- **销售管理**：销售订单、出库管理、客户管理
- **库存管理**：库存查询、出入库记录、库存预警
- **财务管理**：应收账款、应付账款、费用管理
- **报表分析**：多维度数据统计与分析

## 技术栈

- **数据库**：SQLite
- **开发语言**：Python
